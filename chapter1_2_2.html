1.2.2 Wavefunctions for Non-interacting Electrons

This section focuses on the special case of non-interacting electrons, where the Hamiltonian simplifies and the many-electron wavefunction can be constructed from single-particle spin orbitals.

üî∑ Hamiltonian for Non-interacting Electrons

\[ \hat{H}_{\text{non}} = \sum_{i=1}^N \left[ -\frac{1}{2}\nabla_i^2 + v(\vec{r}_i) \right] \]

(1.23) ‚Äî Independent-electron Hamiltonian

The one-electron eigenfunctions \( \psi_\alpha(\vec{r}, \sigma) \) solve this Hamiltonian. The many-body wavefunction is built from these spin orbitals via the Slater determinant:

\[ \Phi = \frac{1}{\sqrt{N!}} \sum_P (-1)^P \psi_{\alpha_1}(P1) \psi_{\alpha_2}(P2) \cdots \psi_{\alpha_N}(PN) \]

(1.25) ‚Äî Slater determinant for antisymmetrization

This construction enforces the Pauli exclusion principle: no two electrons can occupy the same spin orbital. If any two \( \alpha_i = \alpha_j \), then \( \Phi = 0 \).

\[ E_{\text{non}} = \varepsilon_{\alpha_1} + \varepsilon_{\alpha_2} + \cdots + \varepsilon_{\alpha_N} \]

(1.26) ‚Äî Total energy for non-interacting electrons

Example: Helium Atom in Non-interacting Approximation

For the ground state of the non-interacting helium atom (\( N = 2 \)):

\[ \psi_1(\vec{r}, \sigma) = \psi_{1s}(\vec{r})\delta_{\sigma,\uparrow} \quad \text{(1.27)} \]
\[ \psi_2(\vec{r}, \sigma) = \psi_{1s}(\vec{r})\delta_{\sigma,\downarrow} \quad \text{(1.28)} \]

The Slater determinant becomes:

\[ \Phi(1,2) = \frac{1}{\sqrt{2}} \begin{vmatrix} \psi_1(\vec{r}_1, \sigma_1) & \psi_2(\vec{r}_1, \sigma_1) \\ \psi_1(\vec{r}_2, \sigma_2) & \psi_2(\vec{r}_2, \sigma_2) \end{vmatrix} = \psi_{1s}(\vec{r}_1)\psi_{1s}(\vec{r}_2) \frac{1}{\sqrt{2}} (\delta_{\sigma_1,\uparrow}\delta_{\sigma_2,\downarrow} - \delta_{\sigma_1,\downarrow}\delta_{\sigma_2,\uparrow}) \]

(1.29) ‚Äî Antisymmetric wavefunction for He

üß† Summary

  • Non-interacting systems allow simplification using one-electron spin orbitals.
  • The antisymmetrization via Slater determinants enforces Pauli exclusion.
  • The total energy is a sum of single-particle eigenvalues.
  • This formalism is foundational for constructing Kohn‚ÄìSham orbitals in DFT.
